# Hotel Management System Java - Core
A console Based Java Application.

Modules: 
- Analytics
- Billing
- Check In / Check Out
- Club
- Registeration
- Reservation
- Restaurant
- Staff

## Development

```bash
  git clone https://github.com/shreejilucifer/HotelManagementSystemJava.git
  cd HotelManagementSystemJava
  # Compile & Run the Program: 
  javac *.java
  java MainMenu.java
```
